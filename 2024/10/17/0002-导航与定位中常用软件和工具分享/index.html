2.3 BKG Ntrip Client

Ntrip工具

下载地址：BKG Ntrip Client - BNC

The BKG Ntrip Client (BNC) is an Open Source multi-stream client program designed for a variety of real-time GNSS applications.
It was primarily designed for receiving data streams from any Ntrip supporting Broadcaster. The program handles the HTTP communication and transfers received GNSS data to a serial or IP port feeding networking software or a DGPS/RTK application. It can compute a real-time Precise Point Positioning (PPP) solution from RTCM streams or RINEX files. During the last years BNC has been enriched with RINEX quality and editing functions. You can run BNC with GUI as well as in batch processing mode.

BKG Ntrip Client (BNC) 是一个开源的多流客户端程序，专为各种实时 GNSS 应用程序而设计。它主要设计用于从任何支持广播的 Ntrip 接收数据流。该程序处理 HTTP 通信并将接收到的 GNSS 数据传输到串行或 IP 端口馈送网络软件或 DGPS/RTK 应用程序。它可以从RTCM 流或 RINEX 文件计算实时精确点定位 (PPP) 解决方案。在过去的几年里，BNC 已经丰富了 RINEX 质量和编辑功能。您可以使用 GUI 以及批处理模式运行 BNC。

4. 开源代码分享

4.1 RTKLIB

  • 简介: 最著名、应用最广泛的开源GNSS定位库。由东京海洋大学的Tomoji Takasu开发。它支持实时和后处理高精度定位（RTK/PPP），处理多系统、多频率数据。
  • 特点： 功能极其全面，拥有图形化工具（RTKNAVI, RTKPOST等），文档丰富，社区庞大。是学习和研究高精度定位的首选。

Tomoji Takasu版本：tomojitakasu/RTKLIB

应用最广泛的社区版本：rtklibexplorer/RTKLIB

4.2 GPSTk (GPS Toolkit)

  • 简介: 由美国空军研究实验室等机构发起的一个开源C++库，提供了完整的GPS及相关导航系统数据处理工具链。
  • 特点： 更偏向于科研和底层算法，包含了大量关于时间系统、坐标转换、星历计算、数据格式处理的库函数。

非常难得的C/C++ GNSS导航定位开源仓库，里面很多面向对象的设计操作值得学习。

原版本仓库地址（现已归档）：SGL-UT/GPSTk

原作者恢复更新的仓库，拆分为了一个Lib库和一个App库：

  • SGL-UT/gnsstk-apps

  • SGL-UT/gnsstk

4.3 gLAB

  • 简介: 由欧洲航天局（ESA）支持开发的一款用于GNSS数据后处理、分析和教学的高级软件。
  • 特点： 在PPP（精密单点定位）方面非常强大，算法透明，非常适合用于算法研究、教育和精度分析。拥有图形界面。

gLAB is a software tool suite developed under an European Space Agency (ESA) Contract by the research group of Astronomy and Geomatics (gAGE) from the Universitat Politecnica de Catalunya (UPC), is an interactive educational multipurpose package to process and analyse GNSS data.

gLAB performs precise modeling of GNSS observables (pseudorange and carrier phase) at the centimetre level, allowing both standalone GPS positioning and PPP. Every single error contributor may be assessed independently, which, in turn, provides a major educational benefit. gLAB is adapted to a variety of standard formats like RINEX-3.00, SP3, ANTEX and SINEX files, among others. Moreover, functionality is also included for GPS, Galileo and GLONASS, allowing performing some data analysis with real multi-constellation data.

gLAB GitHub仓库：ita-gnss-lab/gLAB

gLAB 官方网站：gLAB Tool Suite — Research group of Astronomy and GEomatics.
